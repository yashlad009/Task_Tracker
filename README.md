# Task_Tracker
This is web application based on improving daily routine , manage flexible task and mentorize yourself to improve efficiency 

## Environment variables (safe for GitHub)

- **Never commit** `.env` (this repo ignores it via `.gitignore`).
- Copy the template and fill in your secrets locally:

```bash
copy .env.example .env
```

- **Required for AI Mentor**:
  - `GEMINI_API_KEY` (kept only in your local `.env`, or set as a hosting provider secret)
